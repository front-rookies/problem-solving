# 📌 문제 설명

프로그래머스 131128 lv.1: 숫자 짝꿍
문제 -> https://school.programmers.co.kr/learn/courses/30/lessons/131128

## 💡 해결 방법

- 처음엔 X문자열과 Y문자열을 배열로 바꾼 뒤에 X배열의 요소와 Y배열의 요소가 같으면 Y배열의 해당 요소를 제거하는 방법으로 접근했음
- 테스트케이스 6~15가 계속 실패해서 인터넷 검색시도
- X문자열과 Y문자열 각각 0~9까지 갯수를 세서 둘 중 적은 숫자를 해당 인덱스에 추가한 후에 repeat메소드를 사용해 answer에 넣어 내림차순 정렬하여 문자로 변환함
- 여전히 테스트케이스 6~15가 계속 실패함
- 중복숫자가 "0000"일 경우 0 중복을 제거하기 위해 String(Number(answer))을 사용하면 6~15에 실패한다고 함
- if (answer.replaceAll("0", "").length === 0) answer = "0"라는 분기를 추가하니 해결됨
